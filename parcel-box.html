PROJECT IOT

Smart Solar Parcel Receiving Box

Project Overview

The Smart Solar Parcel Receiving Box is an intelligent package locker designed to enhance the security and convenience of receiving deliveries. It features real-time notifications via LINE and operates entirely on solar power, making it an eco-friendly and efficient solution for modern parcel management.

📝How It Works

📍Parcel Detection: A IR sensor detects when a package has been placed inside the box.
📍Notification System: The system sends an instant alert via LINE to notify the recipient of the delivery.
📍Solar-Powered Operation: The entire system is powered by a solar panel, ensuring energy efficiency and reliability, even in off-grid locations.

💾Software & Hardware Components

📍ESP32 Microcontroller – Acts as the brain of the system, handling communication, processing sensor data, and sending notifications.
📍IR Sensor (Infrared Sensor) – Detects when a package is placed inside the box.
📍Solar Panel – Provides renewable energy to power the system, ensuring continuous operation without external power sources.
📍Arduino IDE – Used for writing, compiling, and uploading code to the ESP32.
📍C/C++ (Arduino Programming Language) – Programming language used to control the microcontroller.

🏆 Impact & Future Applications

The Smart Solar Parcel Receiving Box is designed to enhance convenience, security, and sustainability in modern package delivery systems. With the rise of e-commerce and online shopping, this innovation helps users manage deliveries efficiently, ensuring that no package is left unattended or lost.
